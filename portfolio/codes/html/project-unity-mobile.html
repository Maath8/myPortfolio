Unity Mobile Game

A mobile adventure built in Unity using C#, where players recover stolen energy cores to save a city. Optimized for Android devices with intuitive touch controls and smooth mobile performance.

About the Project

The city has fallen into darkness after its energy cores were stolen. As a brave hero, you must recover all four cores through challenging levels. Players can complete the full game using touch controls or an Xbox controller, with interactive environments, a responsive UI, and smooth performance on Android devices.

🛠️
How It Was Built

  • Unity Engine with Android Build Support module
  • C# for all game scripts and touch input handling
  • Unity Input System for cross-platform touch gestures (tap, swipe)
  • Canvas UI system scaled for multiple screen resolutions
  • Efficient gameplay interactions implemented using colliders

⚡
Challenges We Faced

  • Adding controller support was tricky due to differences between new-gen and old-gen systems.
  • Implementing Xbox controller support was challenging but rewarding once it worked.
  • Adapting PC game controls for mobile devices took time and careful testing, presenting a fun and valuable challenge.

💡
What I Learned

Developing a mobile game taught me how different the constraints are compared to PC. I learned to optimize textures for mobile, profile performance with Android Logcat, and design interactions primarily for touch. As an extra challenge, I implemented Android controllers and even added full Xbox controller support, which was both difficult and rewarding. This process gave me a deeper appreciation for mobile UX and cross-platform design.
